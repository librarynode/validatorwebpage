Trust questions

FAQ

This section answers the obvious first-pass questions a person or model will ask when evaluating the validator surface.

What does this validator do?

It is a Post Fiat testnet validator operated by @wizbubba. This site exists to package the validator's identity, proof links, and status context in one place.

How do I verify this public key?

Compare the public key shown here against the explorer details page, the machine-readable JSON endpoints on this site, and the domain attestation file in /.well-known/.

What is the change log?

It is the running public log for validator-related updates on this domain. It complements the homepage by giving chronological context instead of only a static summary.

Where do the status metrics come from?

The live status section is backed by the Post Fiat explorer source for this validator. If the explorer is unavailable, the site shows Unavailable instead of filling values from guesses.

How often is this page updated?

Static site content updates when the repo changes. The live status section refreshes from the approved explorer source on page load whenever the source is reachable.

What is this page not?

This is a validator trust surface, not a full explorer and not a general Post Fiat network explainer. It is intentionally narrow: identity, proof, status, and updates.
